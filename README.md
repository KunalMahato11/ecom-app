### Steps to run the app 
1. Start the server  
    - Install dependencies  
    `yarn`  
    - Start the server on localhost  
    `yarn start`  

2. Start the client  
    - `cd client/`   
    - Install dependencies  
    `yarn`  
    - Start the client on localhost  
    `yarn start`
